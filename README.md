# Minecraft 服务器介绍页面

上传即用的 Minecraft 服务器官网系统，无需 composer / npm / 任何构建步骤。

## 功能

- 服务器实时状态（30 秒刷新，Java / Bedrock 双协议）
- 24 小时在线人数统计图表
- 服务器图集展示（分类管理 + 灯箱预览）
- 动态 / 公告发布（富文本 + 封面图 + 置顶）
- 访客留言板（审核制 + 管理员回复）
- 白名单申请系统（提交 + 状态查询）
- 多用户后台（超级管理员 / 内容管理员）
- 功能开关（留言 / 白名单 / 图集等可独立开关）
- 多主题切换

## 环境要求

- PHP 8.1+（需开启 `pdo_mysql`、`gd` 扩展）
- MySQL 5.6+
- 宝塔面板（推荐）

## 安装步骤

1. **上传文件**：将整个项目目录上传到宝塔网站根目录
2. **创建数据库**：在宝塔面板创建一个 MySQL 数据库
3. **运行安装**：浏览器访问 `http://你的域名/install.php`
   - 填写数据库连接信息
   - 自动建表和初始化数据
   - 创建管理员账号
   - 生成配置文件
4. **配置伪静态**：将 `nginx.conf.example` 的内容粘贴到宝塔站点的伪静态配置中
5. **设置定时任务**：宝塔 → 计划任务 → Shell 脚本 → 每 1 分钟执行：
   ```
   php /www/wwwroot/你的目录/cron.php >> /dev/null 2>&1
   ```
6. **访问后台**：`http://你的域名/admin/`

## 目录结构

```
├── index.php          # 前端入口（自动加载当前主题）
├── install.php        # 安装向导（安装后自动锁定）
├── cron.php           # 定时任务（MC 状态查询）
├── config.php         # 配置文件（安装后生成）
├── .htaccess          # Apache 伪静态规则
│
├── core/              # PHP 核心类库
├── api/               # 公开 API
├── admin/             # 后台管理
│   ├── index.html     # 后台面板（CDN 加载，无需构建）
│   ├── api/           # 后台 API
│   ├── js/            # 后台 JS
│   └── css/           # 后台 CSS
│
├── themes/            # 前端主题
│   ├── shared/        # 主题共享资源
│   └── starter/       # 默认主题
│
├── uploads/           # 上传文件
└── cache/             # 文件缓存
```

## 主题开发

在 `themes/` 下新建目录，包含 `index.html` 即可。共享工具库在 `themes/shared/`。

## 安全提示

- 安装完成后 `install.php` 自动锁定
- `config.php`、`core/`、`cache/` 目录已在 Nginx 配置中禁止外部访问
- 首次登录后请立即修改默认密码
